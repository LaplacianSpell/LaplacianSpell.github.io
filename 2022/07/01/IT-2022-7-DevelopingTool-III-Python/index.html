前言：当我们在说学习一门新的编程语言的时候，我们到底在说什么？

一门又一门的编程语言同中有异，异中有同。虽说“实现目的”是我们学习新的编程语言的直接驱动力，可如何高效地掌握一门编程语言，使其为我所用，是个值得我们思考的问题。

我们尝试用过往的经验来对这个问题给出某种意义上的解答。也许你已经修过了《程序设计基础》或是《计算机程序设计基础》等等课程，我们不妨回忆在入门第一门编程语言时的流程。

我们先介绍了一些最简单的数据类型，然后引入了一些加加减减的操作符。然后，我们学习了基本的程序流程控制语句，如顺序语句、条件语句和循环语句，以掌握程序执行的具体流程。然后，我们学习了一些较为方便的数据组织方式，如 C++ 中的数组、结构体、STL 等等，进而用面向对象的思维来思考问题。此外，我们还要了解使用这门编程语言所具有的独特的优势，比如一门编程语言的语法糖，或是使用这门语言能让我们极为便利地完成什么功能。

因此，我们在学习一门新的编程语言时，不妨就仿照着这个思路，将学到的内容按照上述主题进行归类整理。

解释性语言

python的运行和C/C++等编译性语言是不同的.python运行时是在一个特定的shell中运行的，这个shell叫python的解释器.解释器会对命令进行一行一行的解析.

而编译型语言例如C/C++, 需要将源代码（.c, .cpp）转化为可执行程序 (.exe),负责这一过程的就叫编译器；而当项目为了管理方便，将各个模块分散为多个源代码文件的时候，在编译完成之后还有一个将各个文件打包的链接过程。

因为我已经学过并且使用将近三个学期的 C/C++ 了，感觉 C/C++的哲学和 python 的哲学还是很不一样的。我们在 C/C++中可能考虑的更多的概念会是数据的种类，内存，地址等等，正因为地址如此的重要才导致指针在 C++ 中有举足轻重的地位。

python 的思维方式与基本的数据类型

  • 对象是 python 程序中操作与处理的核心元素
  • 对象都有其对应的类型，决定了能够对该对象执行的操作符种类
  • 类型依据其是否“可分”，可以分为标量和非标量，其中标量有 int，float，boolean，None 四种。而内部还有结构的 str 类型等则不是标量。
  • 对象和操作符组成表达式，而表达式等价于一种对象，称为表达式的值
  • 字面量是对象在程序文本中的写法，比如对 -1 的 int 类型，字面量就是 -1；对 -1 的 float 类型，字面量就是 -1.0；对字符串 aabcd，字面量就是’aabcd’,换句话说，字面量就是程序能读懂的语言写的对象

python 的基本数据类型如下

  • int
  • float
  • complex
  • bool
  • string
  • NoneType

Remark I

  1. 在 python 中不需要预先声明和定义变量.C 中定义变量实际上是在开拓一个存储空间
  2. 在 C 中，赋值语句相当于改变存储空间中的数值。
  3. 在 python 中，变量只是一个名字，在 python 中的赋值语句相当于将某个对象绑定到这个名字上。
  4. 在 python 中，int 在需要的时候内存长度会自行增长，同时对两个 int 使用普通除法/会返回一个 float 类型，不会产生 C 系语言的 trunc 问题

  1. python 之所以有这种属性根本在于他是一门动态类型语言，也就是说，python 只在运行到指定位置的时候才进行数据的类型检查。而诸如C，C++，C#等等都需要预先声明所有变量的数据类型，是因为他们的数据类型检查都是在编译阶段进行的，叫做静态类型语言。

  2. 此外，Python 与 C 都是强类型定义语言（安全的语言），也就是说，对于特定对象，他的类型是确定的，而弱类型定义语言则不是这样，举个例子对弱类型定义语言来说'123'与123是一样的。在强类型语言中，不存在隐式的类型转换

作用域

Python 中只有模块（module），类（class）以及函数（def、lambda）才会引入新的作用域，其它的代码块（如 if/elif/else/、try/except、for/while等）是不会引入新的作用域的，也就是说这些语句内定义的变量，外部也可以访问。

Python 的垃圾回收

Python 采用引用计数的方式进行垃圾回收（这再一次体现了 Python 哲学的重要性）.每一个变量的赋值都是在对右边对应的对象贴上新的标签（引用），Python 内部有一个引用计数器会记录贴在这个对象上的标签个数，当标签个数减为零之后，对象立即销毁

因此可以手动使用 del 撕掉标签，来对内存进行释放
