Numpy

Numpy是一个库，里面最常用的对象是ndarray，支持快速的n维数组计算和大量线性代数的操作.

使用Numpy最重要的原因有两个（区别于使用Python内置的array）

  • 支持多维数组（张量）的建立
  • 支持封装好的许多快速的科学运算（线性代数，FFT等等）

ndarray introduction

n-dimensional array（ndarray）是Numpy里面主要的对象.构造的时候可以写为numpy.array()

array attributes

对于如何构造Numpy对象咱们暂且按下不表，我先介绍一个array对象里面的attributes有些什么:

  1. ndim： int，顾名思义就是张量的维数，在Numpy中叫做axes数目.
  2. shape： Tuple，表示张量的形状，如果是二阶张量就是(m,n),m行n列.
  3. dtype： Type，表示张量内部元素的类型.
  4. itemsize: int，表示张量内部元素的大小（Bytes）

array construction

现在我们基于下面这种视角来看高阶张量：比如二阶张量，实际上就是一阶张量，只是元素仍然为同长度的一阶张量.
这个视角十分的重要，也是为什么张量计算如此之快的原因，因为他的本质数据结构是一个连续地址的线性表.并且是用C实现的.
我们称这种视角叫做Numpy的哲学

有下面几种构建方法:

  1. 通过 .array() 这个Method，将元组或者列表转换为相应的张量.
  2. 特殊的构造方法，比如 .zeros()和.ones()用来构造矩阵.
  3. 利用arange来生成array，效果和range类似，只不过返回是array
  4. 对于3.如果我们在其中加入float类型，则有可能出现问题，因为大小不可控.此时需要linspace()来解决，这个和arange唯一的不同在于第三个argument告诉的是区间内取点个数.满足首尾两个数一定取到.
