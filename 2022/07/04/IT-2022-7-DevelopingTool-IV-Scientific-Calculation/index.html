Numpy

Numpy 是一个库，里面最常用的对象是 ndarray，支持快速的 n 维数组计算和大量线性代数的操作。

使用 Numpy 最重要的原因有两个（区别于使用 Python 内置的 array）

  • 支持多维数组（张量）的建立
  • 支持封装好的许多快速的科学运算（线性代数，FFT 等等）

ndarray introduction

n-dimensional array（ndarray）是 Numpy 里面主要的对象。构造的时候可以写为 numpy.array()，括号内可以选择的参数有 List，Tuple等

array attributes

对于如何构造 Numpy 对象咱们暂且按下不表，我先介绍一个 array 对象里面的 attributes 有些什么：

  1. ndim： int，顾名思义就是张量的维数，在 Numpy 中叫做 axes 数目。
  2. shape： Tuple，表示张量的形状，如果是二阶张量就是 (m,n),m 行 n 列。
  3. dtype： Type，表示张量内部元素的类型。
  4. itemsize: int，表示张量内部元素的大小（Bytes）
  5. size： int, 表示张量内部元素的个数

array construction

现在我们基于下面这种视角来看高阶张量：比如二阶张量，实际上就是一阶张量，只是元素仍然为同长度的一阶张量。
这个视角十分的重要，也是为什么张量计算如此之快的原因，因为他的本质数据结构是一个连续地址的线性表。并且是用 C 实现的。
我们称这种视角叫做 Numpy 的哲学

有下面几种构建方法：

  1. 通过 .array() 这个 Method，将元组或者列表转换为相应的张量。
  2. 特殊的构造方法，比如 .zeros()和 .ones()用来构造矩阵。
  3. 利用 arange 来生成 array，效果和 range 类似，只不过返回是 array
  4. 对于 3.如果我们在其中加入 float 类型，则有可能出现问题，因为大小不可控。此时需要 linspace()来解决，这个和 arange 唯一的不同在于第三个 argument 告诉的是区间内取点个数。满足首尾两个数一定取到。（如果不想取可以设置第四个参数 endpoint 为 False）

Matplotlib

Matplotlib 是一个绘图神器工具，其大量的思想来源于 Matlab

Pyplot and Fundamental Function

首先，我们来介绍其的一个绘图子库，from matplotlib import pyplot，我们可以调用相关的函数来绘图。

Matplotlib 将你的数据绘制在 Figure 上。每个 Figure 可以包含一个或多个 Axes，Axes 是一个可以指定 x-y 坐标并绘制数据的区域 (也就是说我可以在一个 Figure 里面绘制多张 Axes)。创建带有坐标轴的图的最简单方法是使用 pyplot.subplots 然后我们可以使用 Axes.plot 在坐标轴上绘制一些数据。大部分的绘图方法被定义在 Axes 类和它的成员函数上面。常用的成员函数有 set_title(), legend(), set_ylabel(), set_xlabel(), xaxis(), yaxis(), plot(), scatter() 这些

注意不要混淆 Axes 和 Axis 前者是一个可以指定 x-y 坐标的区域，后者指的是一个坐标轴。
