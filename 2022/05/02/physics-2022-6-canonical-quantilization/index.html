力学量随时间的变化率与泊松括号

我们可以借助Hamilton正则方程导出对于某一力学量的随时间变化的变化率

对于任何力学量 AAA ，我们都可以将其表述为广义坐标 qqq ，广义动量 ppp 以及时间 ttt 的函数，因此有：

dAdt=∂A∂t+∂A∂qiqi˙+∂A∂pipi˙\frac{dA}{dt} = \frac{\partial A}{\partial t} + \frac{\partial A}{\partial q_i}\dot{q_i}+\frac{\partial A}{\partial p_i}\dot{p_i} dtdA​=∂t∂A​+∂qi​∂A​qi​˙​+∂pi​∂A​pi​˙​

此处采用了Einstein Notation
代入Hamilton正则方程

∂H∂pi=qi˙ ; −∂H∂qi=pi˙\frac{\partial H}{\partial p_i} = \dot{q_i} \ ;\ -\frac{\partial H}{\partial q_i} = \dot{p_i} ∂pi​∂H​=qi​˙​ ; −∂qi​∂H​=pi​˙​

我们得到

dAdt=∂A∂t+(∂A∂qi∂H∂pi−∂A∂pi∂H∂qi)\frac{dA}{dt} = \frac{\partial A}{\partial t} + (\frac{\partial A}{\partial q_i}\frac{\partial H}{\partial p_i} - \frac{\partial A}{\partial p_i}\frac{\partial H}{\partial q_i}) dtdA​=∂t∂A​+(∂qi​∂A​∂pi​∂H​−∂pi​∂A​∂qi​∂H​)

记括号内的为

[A,H]=∂A∂qi∂H∂pi−∂A∂pi∂H∂qi[A,H] = \frac{\partial A}{\partial q_i}\frac{\partial H}{\partial p_i} - \frac{\partial A}{\partial p_i}\frac{\partial H}{\partial q_i} [A,H]=∂qi​∂A​∂pi​∂H​−∂pi​∂A​∂qi​∂H​

我们称其为泊松括号。
我们可以进而得到广义坐标和广义动量之间的正则相容关系

[qi,qj]=0 ; [pi,pj]=0 ; [qi,pj]=δij[q_i,q_j] = 0 \ ; \ [p_i,p_j] = 0 \ ; \ [q_i,p_j] = \delta_{ij} [qi​,qj​]=0 ; [pi​,pj​]=0 ; [qi​,pj​]=δij​

Dirac注意到了，利用某种代数（待补充）可以将对正则变量的微商运算转化为对力学量算符的运算

[A,B]→1iℏ[A^,B^][A,B] \rightarrow \frac{1}{i\hbar}[\hat{A},\hat{B}] [A,B]→iℏ1​[A^,B^]

进而可以得到关于相容正则变量对应的算符的相容性.
