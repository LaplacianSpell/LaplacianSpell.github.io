咖啡是苦的

发表于2023-05-27|更新于2023-09-25
|字数总计:3.4k|阅读时长:9分钟|阅读量:

咖啡是苦的

透明的、高饱和度的、炎热的夏天。没有风的下午，干热的空气静止下来，仿佛凝结成一团一团的。

咖啡馆里，一个白色的薄薄的小展牌上用黑色和浅蓝色马克笔以一种很解暑的方式写着“新品！冷萃咖啡，柑橘、茉莉、柠檬风味”几个大字。很难不令人心血来潮想要尝尝。我咽了咽口水。

“我要一杯这个。”

小小的 330ML 竟然花掉了我 28 块软妹币！穿着白色厨袍的服务生笑盈盈地从冰箱里把那支装满咖啡的瓶子拿到我手里时，我的心和我的手掌大概是同一个温度。

凑到瓶子跟前，鼻子甚至能呼吸到经瓶壁降了温的空气，盯着渗水的、雾蒙蒙的瓶壁向里面看了又看，里面的深褐色液体晃来晃去，显得有些不安分。我等不及了，拧开盖子便是一口，甚至想要一饮而尽。

没有预料之中那种平时常喝的拿铁或者卡布奇诺的味道，整个口腔随即被某种从未尝过的苦与酸充盈着，一直延续到喉咙，那种味道是那样的深刻，仿佛一只大手紧紧地攥住了我的舌头。我闭上了眼，以免把此时此刻的味觉就这样写在脸上好生让人笑话。

大概是拿铁这种掺了杂的东西让我产生了对这个东西的错误印象，上网一查才知道，咖啡本来的味道就是又酸又苦的，不同品种的咖啡酸的还不一样，还有明亮的酸味，暗淡的酸味，悠长的酸味等诸多酸味之说。看完科普，我旋即又拧开了那瓶黑咖啡，这才看见标签上写着埃塞俄比亚阿拉比卡云云，是高品质的咖啡豆。

不过我不懂这些，我只知道那一天的咖啡是苦的，冰块是凉的，那一天的太阳是白的，天空是蓝的。

我为什么喜欢理论物理和技术

今晚考完群论，去拿东西时被说到“我想象中的觉得你会去做类似于计算、模拟之类的方向” ，想着考完了一门也干完了今天要干的事情，便绕着学校，边走边胡思乱想。

我觉得学习理论和技术对应两种极其单纯的快乐。学习和研究技术像拆东西、搭积木；而学习和研究物理像大海边捡贝壳，像某种大型 RPG 解谜游戏里面找某种带有魔法的碎片。前者所具有的明晰的方向感和对想象力的简单要求就像玩游戏一样会带来快速的快乐反馈；而后者则以一种长周期的形式存在，在后者那样的情况下，你不知道未来会发生什么，你只能一边勇敢地行走一边笨拙地在沙滩上挖一个又一个的坑，尝试着拼凑不同时间、在不同地点发现的碎片；但是搭积木终究是搭积木，就像吃零食很快乐但是在我看来终究不能代替正餐一样；而捡贝壳却能发现珍珠，找到能拼在一起的魔法碎片能够…嗯…总能够做些什么很神奇的东西吧？

特别感谢科协技术组，在里面就突然间感觉自己之前随便玩玩的脏东西好像还挺有意义的，其实说实话自己在技术上也很小白，只是当时捡着一些电子系的边角料和培训多学了一点点而已。真的很不值得一提。

我自认为这学期学群论是很合适的，一边学一边回想各种之前有些不理解的东西，同时加深了对现在的许多东西的理解。群论就好像精灵一般在各个地方时隐时现，这种感觉就像我说的那样，你突然间发现某些看似不相关的东西在某种神秘的数学框架下统一了起来，此时，NPC 出现并告诉你：“恭喜你，你集齐了魔法碎片！”，你会惊呼： “vola！真妙啊！”。是啊，确实很妙。这也促使我认识到知识本身永远都是美的，让你觉得不美的永远是“课程”，是“GPA”，是“某某老师”，是“考试”，甚至可以引申一下，是以后做科研可能会遇见的烂导师（？），坏的环境（？）…而不是知识本身。

于是在这种冲动下这学期甚至报名了天文学辅修学位，大概到毕业能收获一个满满的、充满各种故事成绩单了，就是虽然成绩不是那么的好但似乎也还过得去的样子（笑）

突然感觉自己做的一切都很兴趣使然，似乎永远都有一些乱七八糟的好奇心，怎么爽怎么办的哲学好像看起来很危险的样子，而且这样搞之后推研不知道会遇到什么麻烦，不知道现在这样是好事还是坏事呢（）

走着走着，又想回了群论上面，突然脑子一闪发现考试自己有个地方少做了一步，气的连拍了几下大腿。

就这样，突然间梦就醒了，我站在紫操的跑道上，晚上风有点大。

断点

断点（Break Point）是计算机科学的一种 debug 方式。

不要忘了给你好的状态设置断点，也即总结保持好的状态所需要的一些条件。

各种疾病和事件等真是不可预料。生病回来忘记了当初自己是怎么高效率完成各种任务的不免有点可惜。花时间再探索一遍经验实在是不能忍受。

会成功吗？

期待你下一个学期与高年级同学达到热平衡 (笑)

– Prof. zzxianyu

不知道该说些什么，感觉胸口热烘烘的，如果，我是说如果，自己以后真的选择从事这样的职业，那么也千万不要忘记以同样的热情去鼓励你的学生。

Fermi Problem

今天看 Kurt Gottfried, Tung-Mow Yan 写的 QM，这本书里面佐证理论的实验很现代，我很喜欢，提到了一个 PRL 上面的工作 (Phys. Rev. Lett. 70, 2359(1993)) 让我有了一种想要在近代物理实验里面搭建一个离子阱囚禁 $\mathbf{Hg}^{+}$ 之后验证该实验的冲动。

然后不由得想到了 Fermi 这位理论和实验兼修的大方之家。物理学细化到今天，大概想成为 Fermi 那样的人还是很困难的吧，不知道自己能不能做一点 non-trivial 的工作呢？

又看到 Fermi Problem 里面一些很有意思的东西，我觉得这种问题的解决是有灵性的，让我想到了 zzxianyu 老师估计宇宙中的电子数目。

我不喜欢变成 nerd，我希望自己变得有灵性，aggressive but calm，从丝绒中感受力量。

不得不说，实验也很有意思啊，就是那种搭建一些属于自己的小东西的感觉，像平时玩的玩具（笑），自己想学一些技术也是 motivated by 这种感觉吧…

尽管说了很多遍，我还是要说：

有意思的东西实在是太多了

不管怎样，先行动起来再说吧！

Ego ideal

在上大学之前，我对“自己想成为什么样的人”这一问题的答案大概是很模糊的，在 18 年的时光里（甚至只是一小部分的时光里，更准确来说是高中），我大概只是把成为 B 市 T 大的一员作为自己的人生目标。

上大学之后，我开始既小心翼翼地而又大胆地做各种尝试。在不断地学习和自我学习当中，那个潜藏在我心底的模糊画像逐渐变得清晰；在随机的碰壁和痛苦以及成功与喜悦之中，我顿悟了许多事情。自己经常无病呻吟的一事无成，倒也没有那么的不堪；站在我大学生活第三年末的这个时刻突然回头，发现自己原来走了这么远啊！

生活是一个 poisson 过程，生活是一个外场下的随机行走。

所以，我想成为怎么样的人呢？

  1. 理论物理学家

    这个词当然还是太过于宽泛了，不过自己现有的认知水平似乎只能细化到这个程度，再细一点大概是高能方向，Quantum Gravity，大概是某种求知的渴望。

    这个理想是我的初心。

  2. 开源软件开发者

    关于这个，我可以理出一条清晰的成长路径，我第一次接触到编程是在 2020 年的秋季学期，在这之前自己并没有这方面的任何知识，秋季学期学习了 C，春季学期学习了 OOP C++，小学期在 Arduino 上嵌入式开发了一堆垃圾，三天速成 Python 并用 Pytorch 在 Herr 盛的帮助下糊了一个卷积神经网络，用 C++ 的 STL 快速糊了一个学生档案管理系统，在读写文件的 bug 没有 de 掉的情况下胡乱交上去并拿了一个 A。不过这些都不重要，我觉得我真正的开源软件之路是从三系联合暑期培训开始的。

    从暑期培训，我第一次接触到 Linux，git 等等相关的工具，接触到了许多前端后端知识，接触到了 C# 等，不过也只是泛泛的学了一点，当时只是觉得很新奇，同时比之前对于一些东西的理解确实深刻了不少，从未想到这些东西会在这两年里慢慢让我对开源软件和计算机有了与之前大不一样的认识。

    之后沉沦了一年，在电子系物理系课程的轰炸下 arduously living，在转系过去之后迎来了开源人生的第二个转折点，无系的暑期培训以及 实验物理的大数据方法。

    在这次暑期培训时自己对许多东西已经有了一粗浅的了解，很多工具自己也已经开始经常使用了。相当于是训练了一下，最重要的还是实验物理的大数据方法，这门课让我遇见了很好的一位老师，工程物理系的续本达老师，还遇到了一位很好的学长，物理七字班的 Berrysoft 学长，还在他的“怂恿”下加入了 TUNA，也即 清华大学开源软件协会，并且认识了很多很好的同学，其中很重要的一个是物理系零字班的 yfzhao。这个暑假收获颇丰，是码力大为长进的一个暑假，是学会了极多东西的一个暑假，是结识了很多优质友谊的暑假…我会永远记得那个夏天，那整整四周烈日炎炎的下午，那些和续老师以及几个同学有说有笑的走进呼呼吹着冷风的法图地下的日子。在这门课上，品味到了真正的**技术之美 (Art of Technology)**。

    之后便是在物理系的第一学期，第一学期过的有点痛苦。不过加入了物理系科协，这是我的第三个转折点。首先是发现 berrysoft 是科协的老学长，之后又发现 续本达 老师是科协技术组的指导老师，是 TUNA 的顾问，yfzhao 也在科协技术组，同时认识了 mhliu，技术组的组长，现在（2023 年春，科协 27 届）是科协主席。同时知道了科协有个 大服务器：Sexier；在科协作为主讲开了两次讲座，一次是 LaTeX 的，一次是 Linux 的，总之很快乐，“友谊是魔法”…

    之后的事情便不用多说了，在科协的正常工作之中不断探索，不断学习各种技术，并树立了一些志向。

    • 成为 Debian 社区的开发者
    • 学习 类 Unix 系统上的软件开发
    • 努力学会处理天文，高能数据，比如 CERN ROOT 以及高能理论的 Feyn Calc 等，现在这块的技术栈是搭起来了，就差学专业知识了
    • 本科阶段怎么着都想参加一次中科院的开源之夏 OSPP，大概是瞄准续老师的会带的项目，都是挂靠在 TUNA 下面的，不过自己先熟悉一下 Debian 社区的开发相关知识，同时用一段时间的 Debian 先，并且要熟悉起来前/后端的一些东西。
    • …

    这些都是业余爱好，但是足够的有趣

  3. 成为一名电吉他手，能够闲暇之余练一练琴，演奏自己喜欢的音乐，同时学会混音，学会调节自己需要的音色。

Naive Laplace 还有很多东西需要学习，希望大家多教教我！
