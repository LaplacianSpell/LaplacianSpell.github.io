IT-2024-7-Diffgeo

发表于2024-08-07|更新于2024-09-12
|字数总计:1.6k|阅读时长:6分钟|阅读量:

又是许久没有在博客上写东西，突然动笔写点什么，倍感亲切，大抵人的灵魂总奢望着向世间输出点什么，以欺骗自己平庸的事实。

本文是 Diffgeo.m，的简单介绍，甚至可以理解为对原文的翻译。

Diffgeo 是在给定度规张量的基础之上进行计算的程序包，且与 xAct 相比，内置了许多方便物理学家使用的函数，由于程序包的封装程度并不高，因此可以较好的在 MMA 原有代码风格基础上继承使用。

设置度规张量

设置度规张量有两种方法，一种是直接给定线元

注意

  1. ds2 以及 dd[ ] 方法内置为线元方法
  2. 可以通过函数 Cartesian[{x1, x2, x3, x4}]，快速指定给定坐标（这里是 x1,x2,x3,x4）的 Euclidean 度规. 例如共形平直时空 Ω[x1, x2, x3]^2 Cartesian[{x1, x2, x3}].
  3. 可以通过 unitSphere[{θ1, θ2, φ}] 快速指定给定坐标（顺序很重要，θi 是 polar angle，而 φ 是 $2\pi$ 的 azimuthal angle）的 sphere-metric

另一种是以矩阵形式给定度规

注意

  1. metric，coord为内置变量。
  2. 此时需要按照给定的坐标列表写出度规的形式。
  3. 对于不在对角线上的度规，适合在对角之后，直接指定,例如 metric[[1,2]] = metric[[2,1]] = Cos[\[Theta]].
  4. metricsign 即度规张量的行列式，需要在计算前设置，如果不设置则 diffgeo 会自行判断.

构造张量

可以使用 tensor = zeroTensor[rank] 构造有 rank 个指标的张量。然后使用 tensor[[t, r, r]] = value 的方式填充非平凡的部分。注意，diffgeo.m 的逻辑是你使用 lower 的时候，就把你的指标看作是逆变指标；你使用 raise 的时候，就把你的指标看作协变指标，一切由你自己控制。

展示张量

展示张量的分量，可以使用三种方法，一种是指标序号，一种是指标名称，最后一种是 display 函数

display[tensor] 函数会以列表的形式列出张量的所有非零分量以及对应的指标

张量操作

  1. 指标升降：使用 raise[tensor, {index1, index2, index3,...}] 来将从左到右 第 index1, index2, index3 个张量指标升高指标，{index1, index2, index3,...} 缺省为将所有的指标上升。lower 同理.

注意，只有下方的指标能够上升，也只有在上面的指标能够下降，所处位置不对的指标强行上升和下降会得到错误的结果！！！

  1. 张量外积：**，例如 Riemann**Riemann 给出一个(2,6)-张量 $R_{abc}^{\ \ \ \ d}R_{efg}^{\ \ \ \ h}$.

  2. 张量内积：.，实际上是 MMA 内置的矩阵（张量）乘积，且不管指标，就是 naive 的乘法.

  3. 张量指标缩并：contract[tensor, {index1,index2}, {index3,index4}, . . .]，缩并张量 tensor 的指标 index1 与 index3，index2 与 index4 等，如果没有指定指标，就缩并前两个指标。注意，所有参与缩并的指标都默认是下指标。如果想要两个张量缩并，可以先外积，再缩并.

  4. 协变导数：covariant[tensor, {告诉 cov，tensor 的指标位置在哪儿}]，例如 Riemann 需要告诉 covariant 指标位置为 {down, down, down, up}. 因为需要计算 Christofell，缺省默认全部为 down，作用后协变导数的指标为第一个。

  5. Lie导数: Lie[vector, tensor, {告诉 lie，tensor 的指标位置在哪儿}]，这里的 vector 必须为上指标，与协变导数一样，也要告诉指标的位置。

  6. 张量模：norm[tensor]，给出某一张量所有指标与相同张量上下缩并后的结果，注意，输入的张量 tensor 必须为下指标，否则计算 norm 的结果不可信。

其他张量操作

微分形式

tensorToForm[stardf] 可以将全反对称张量转换为以d(**)写出来的微分形式，formToTensor 则可以转换为张量。除了构建全反对称的张量， diffgeo 还可以使用外代数构建微分形式。注意，dd[r ,t] 即 ddr ^ ddt,而 dd[t,\[Theta],\[Phi]] 意思就是 dt ^ d\[Theta] ^ d\[Phi].

内置函数

  1. wedgePower[form , n]：求 $(\wedge \omega)^n$ 的结果
  2. volumeForm，体元，$\sqrt{g} dx^n$，等价于 LeviCivita 张量

超曲面

构建某一坐标为常数对应的超曲面 hypersurface[r]，该方法可以指定方向；general的 hypersurface[ { y1, y2, ... }, { x1 -> x1func of yi, x2-> x2func, ... } ]，第一个为超曲面坐标，第二个为嵌入坐标。例如这里的 hypersurface[{r, \[Theta], \[Phi]}, {t -> r^2}] 类空 Cauchy 面。

自动计算

  • HScoord (list of coordinates on the hypersurface);
  • HSmetric (induced metric);
  • HSinverse (inverse induced metric);
  • HSunitNormal (unit normal co-vector, with index down);
  • HSpullback (derivative of ambient coordinates with respect to hypersurface coordinates; matrix used to pull an ambient covector back onto the hypersurface; first index is a lower hypersurface index, second index is an upper ambient index);
  • HSprojector (rank-2 tensor that projects orthogonally onto the hypersurface, with both indices down);
  • HSextrinsic (extrinsic curvature tensor, in terms of the hypersurface coordinates, with both indices down);
  • HSextrinsicTrace (trace of the extrinsic curvature);
  • HSmeanCurvature (extrinsic curvature trace divided by hypersurface dimension).
